Kansiosta Dokumentit löytyy ohjelman alustava, määrittelyvaiheen luokkakaavio,
pit- ja cobertura raportit sekä tuntikirjanpito.

Kansio OhHa on javaprojektin pääkansio.

Pelin ominaisuudet so far:
Pelaaja voi liikkua pelikentän rajoittamana mihin vain. Pelaaja kuolee jos hirviö osuu pelaajaan.
Hirviö kuolee jos se osuu pelaajan voimakenttään. Voimakentän pelaaja voi laukaista kerran kierroksessa.
Pelaaja saa hirviöstä 10 pistettä ja joka 5. kierros uusi hirviö ilmestyy. Pelaajan kuoltua tulostetaan
lopullistet pisteet. Toistaiseksi pelaajalla on mahdollisuus voittaa peli mutta se tulee muuttumaan.
